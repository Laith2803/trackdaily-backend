# TrackDaily Backend

## Projektbeschreibung
TrackDaily ist eine Webanwendung, mit der Nutzer eigene tägliche Tracker für Gewohnheiten und Aktivitäten anlegen können.

Beispiele:
- Lesen
- 10.000 Schritte erreichen
- ins Gym gehen
- Wasser trinken

Nutzer sollen täglich festhalten können, was sie gemacht haben. Später soll die App monatliche und jährliche Auswertungen bereitstellen.
